🚀 What is FrankPass Pro?

FrankPass Pro is our exclusive browser extension that brings deterministic, stateless password generation right into your browser. Instead of switching tabs or copying/pasting from the website, the extension detects the platform you are on, generates your unique password, and autofills the input field instantly in 1-Click.

💻 How to Install (Developer Mode)

Currently, the extension is in review for the Chrome Web Store. For early access, install it via Chrome Developer Mode:

  1. Download the FrankPass_Pro_v2.3.0.zip file.
  2. Extract the contents into a folder (e.g., on your Desktop).
  3. Open Chrome and navigate to chrome://extensions/.
  4. Enable Developer mode in the top-right corner.
  5. Click Load unpacked and select your extracted folder.
  6. The extension is now installed! Pin it to your toolbar for 1-click access.

🔑 Using Your License Key

To unlock the pro features, you need a valid License Key:

  1. Click the FrankPass icon in your browser toolbar to open the Setup Wizard.
  2. Enter the License Key you received (For testing purposes, you can use: TEST-TRIAL).
  3. Create a 4-digit PIN lock. This prevents anyone from generating your passwords if they sit at your computer.
  4. Choose a beautiful color theme for your local profile.

⚙️ Generating Passwords (1-Click Autofill)

  1. Visit any login page (e.g., netflix.com or facebook.com).
  2. Click the FrankPass extension icon. It will automatically detect Netflix.
  3. Select the country/region, enter your Username and Secret Key.
  4. Click Generate Password.
  5. The extension will instantly inject the 64-character password straight into the login form. No clipboard tracing, no typing, no DB logs.

⚠️ Important Security Notes

  • The Web Platform (frankpass.com) and the Extension share identical cryptographic cores. They will always generate the exact same password for the same inputs.
  • Never forget your Secret Key. If you modify it even slightly, your generated passwords will permanently change.
